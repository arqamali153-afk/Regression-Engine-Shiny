# Executive Regression Engine

A professional, interactive data analysis dashboard built with R and Shiny. This tool automates complex statistical workflows, providing end-to-end regression modeling, diagnostic auditing, and executive business forecasting.

## 🚀 Live Demo
Access the live interactive sandbox here: [Live Interactive Dashboard](https://arqamresearch.shinyapps.io/Executive-Regression-Engine/)
## 📊 Technical Scope
- **Automated Diagnostics:** Built-in Shapiro-Wilk and Breusch-Pagan testing to validate regression assumptions before modeling.
- **Advanced Regression:** Robust linear modeling with support for log-transformations and interaction terms.
- **Automated Reporting:** Integrated RMarkdown to compile client-ready HTML reports of all statistical findings.
- **UI Architecture:** Custom CSS-styled dashboard designed for intuitive business decision-making.

## 🛠 Tech Stack
- **Languages:** R
- **Frameworks:** Shiny, RMarkdown, ggplot2, plotly, bslib
- **Statistical Libraries:** car, nortest, lmtest
